# Your Energy

A fitness web application SPA built with Vanilla JavaScript and Vite.

## 🚀 Live Demo

[View Live](https://your-username.github.io/your-energy/)

## 📌 About

**Your Energy** is a single-page application for browsing fitness exercises. Users can:
- Browse exercises filtered by Muscles, Body parts, or Equipment
- Search exercises by name
- View detailed information in a modal window
- Add/remove exercises to Favorites (saved in localStorage)
- Rate exercises
- Subscribe to email newsletter

## 🛠️ Tech Stack

- Vanilla JavaScript (ES Modules)
- Vite (build tool)
- CSS3 with custom properties
- REST API: [Your Energy API](https://your-energy.b.goit.study/api-docs/)

## ⚙️ Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
